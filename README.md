# kalkulator-reszty
wyliczanie reszty

# Opis projektu
Aplikacja zaczyna się od ekranu logowania. Po zalogowaniu użytkownik wpisuje kwotę zakupu oraz kwotę, którą zapłacił. Program automatycznie sprawdza, czy zapłacona suma pokrywa koszt zakupu. Jeśli zapłata jest większa, aplikacja pokazuje, ile trzeba wydać reszty. Jeśli zapłacono za mało, informuje, ile brakuje do pełnej kwoty. Gdy zapłata jest dokładna, aplikacja potwierdza, że wszystko się zgadza.

# Główne funkcje
-Logowanie użytkownika do systemu (bez opcji rejestracji)
-Wprowadzanie kwoty zakupu
-Wprowadzanie kwoty zapłaty
-Automatyczne sprawdzanie, czy zapłata pokrywa koszt zakupu
-Wyświetlanie reszty do wydania, gdy zapłacono za dużo
-Wyświetlanie kwoty do dopłaty, gdy zapłacono za mało
-Potwierdzenie poprawnego rozliczenia, gdy zapłata jest dokładna
